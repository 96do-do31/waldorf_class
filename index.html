발도르프: 자기소개 웹 페이지 만들기

1. 교육 철학과 목표

본 수업은 발도르프 교육의 핵심 가치인 전인적 성장, 자기 표현의 자유, 그리고 공동체적 상호작용을 디지털 학습 환경 속에 담아내는 것을 목표로 합니다. 기술 습득 자체를 목적으로 하기보다는, 학생이 자기 자신을 진정성 있게 표현하고, 동료들과의 관계 안에서 창의적으로 협력하는 경험을 쌓는 데에 중점을 둡니다.
이를 통해 학생들은 단순한 코딩 기술을 넘어서, 디지털 매체를 통해 자기 이해와 타인과의 소통을 동시에 배우게 됩니다.

2. 학습 과정과 연속성

본 수업은 점진적이고 통합적인 학습 과정을 따릅니다. 단계별로 서로 긴밀히 연결되어 있으며, 매 단계에서 학생들은 자기 자신을 표현하는 방법을 확장해 나갑니다.

  1. 기초 이해 – HTTPS 통신
    먼저 안전한 인터넷 소통의 원리를 배우며, 디지털 세계 속에서 신뢰와 투명성이 어떻게 구축되는지를 이해합니다. 이는 발도르프 교육에서 강조하는 ‘세계에 대한 경외와 존중’을 디지털 차원으로 확장하는 과정입니다.
  2. 표현의 시작 – HTML
    순수 HTML을 이용하여 자기소개 페이지를 제작합니다. 학생들은 자기 자신을 언어와 구조로 표현하는 기본기를 익히며, 이는 마치 손글씨로 자기 소개를 적는 듯한 직접적이고 솔직한 표현 경험이 됩니다.
  3. 아름다움과 조화 – CSS
    CSS를 활용해 색채, 형태, 균형을 적용합니다. 발도르프 교육에서 예술 활동을 통해 감각을 기르는 것처럼, 학생들은 디지털 환경 속에서도 미적 감수성과 조화로움을 경험합니다.
  4. 생명 불어넣기 – JavaScript
    JavaScript를 통해 자기소개 페이지에 움직임과 상호작용을 더합니다. 이는 정적인 작품에 ‘생명’을 불어넣는 과정이며, 발도르프에서 강조하는 의지와 창조성의 실천으로 이어집니다.
  5. 공동체 속 상호작용 – 학년 웹 페이지 제작
    그룹 활동으로 각자의 자기소개 페이지를 하나의 학년 페이지로 연결합니다. 이를 통해 학생들은 서로의 개성과 표현을 존중하며, 함께 협력해 공동체적 결과물을 만들어냅니다. 이는 발도르프 교육이 지향하는 사회적 기술과 상호 존중의 학습을 디지털 방식으로 실천하는 것입니다.

3. 기대되는 교육 효과

  • 자기 이해와 자기 표현: 자기소개 페이지를 만들며 스스로를 성찰하고 표현하는 능력을 기릅니다.
  • 예술적 감수성 확장: CSS를 통해 조화와 균형, 아름다움을 경험합니다.
  • 창조적 의지와 생동감: JavaScript로 상호작용을 구현하며, 자신의 아이디어를 실제로 실현하는 힘을 배웁니다.
  • 공동체적 협력: 학년 페이지 제작을 통해 친구들의 작품을 존중하고 함께 협업하는 경험을 쌓습니다.

👉 이 수업은 단순한 프로그래밍 수업이 아니라, 발도르프 교육 철학에 기반한 전인적 디지털 예술 활동입니다. 학생들은 자기 자신을 표현하고, 타인과 연결되며, 기술을 통해 세계와 조화롭게 소통하는 힘을 기르게 됩니다.
